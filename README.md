# How to install

**First :**

```bash
Open CLI (Command Line Interface)

Clone Repo

git clone https://github.com/aryejfa/employee-management.git
```

```bash
cd employee-management
```

```bash
npm install
```

**Second :**

```bash
Run apps

ng serve --open

Open apps by localhost
http://localhost:4200

Access Login
Username : aryejfa@gmail.com
Password : 123456
```
